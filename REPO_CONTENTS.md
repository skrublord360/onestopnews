# Public Repo Contents

This folder is intended for the public OneStopNews repository.

## Include These Files

```text
public-repo/
├── README.md
├── Project_Requirements_Document.md
├── TECHNICAL_ARCHITECTURE.md
├── REPO_CONTENTS.md
└── assets/
    └── screenshots/
        ├── desktop-home.png
        ├── mobile-home.png
        ├── architecture-overview.svg
        └── data-flow.svg
```

## Do Not Include

- backend source code
- source ingestion logic
- classification rules
- summarization logic
- data cache files
- environment variables
- deployment secrets
- private execution plans

## Purpose

The public repo shows the product, live site, requirements, screenshots, and public-safe architecture. It should help visitors understand OneStopNews without exposing the core implementation.
